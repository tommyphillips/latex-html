3.1.4 Latex Commands

commands tell latex what to do at a particula rpoint in a document commands can take the following basic forms:

  • Modal-vs-Token
  • Control Words
  • Control Symbols
  • Character Sequences
  • Internal Commands
  • Short and Long Commands
  • Declarations
  • Arguments
